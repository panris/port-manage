## 📝 用户使用步骤

### Mac用户
1. 解压 `PortManager-v1.0.0.zip`
2. 进入 `release` 文件夹
3. 双击 `PortManager.command`
4. 首次运行可能需要右键→打开（macOS安全限制）
5. 浏览器自动打开 http://localhost:9527

### Windows用户
1. 解压 `PortManager-v1.0.0.zip`
2. 进入 `release` 文件夹
3. 双击 `PortManager.bat`
4. 如有安全提示，点击"仍要运行"
5. 浏览器自动打开 http://localhost:9527

- **后端**: Spring Boot 2.7.18 + Java 11
- **前端**: 原生JavaScript + HTML5 + CSS3
- **端口扫描**: lsof (macOS) / netstat (Windows)

## 🔧 高级使用

### 命令行启动（跳过脚本）

#### Mac/Linux
```bash
cd release
java -jar port-manage-web.jar
```

#### Windows
```cmd
cd release
java -jar port-manage-web.jar
```

### 自定义端口
```bash
java -jar port-manage-web.jar --server.port=8080
```

### 查看日志
```bash
java -jar port-manage-web.jar > app.log 2>&1
```
